## 🛠 Technical Arsenal

This project leverages a modern, lightweight stack to ensure zero-latency interactions and fluid 3D rendering:

* **Core Framework:** React + Vite
* **Language:** TypeScript
* **Styling:** Tailwind CSS & shadcn/ui
* **Animations:** Framer Motion (Spring Physics)
* **3D Rendering:** React Three Fiber 
* **Additional Tools:** Python, Supabase, Vercel, OpenCV, YOLO

## 💻 Local Development

To run this project locally, ensure you have Node.js installed on your machine.

### 1. Clone the repository
```bash
git clone [https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git)
cd YOUR_REPO_NAME
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```

The application will be available locally at `http://localhost:5173`.

## 🌍 Deployment

This architecture is optimized for edge deployment. It is designed to be seamlessly deployed on platforms like **Vercel** or **Netlify**. Simply connect the repository, and the build command `npm run build` will automatically compile the optimized static assets.
